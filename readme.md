   一套火车的调度和车票出售的集成系统
   
   功能：
   ①账号、角色的管理
   ②车厢型号管理
   ③车组管理（即将不同数量不同型号的车厢拼成车列）
   ④车次管理（使用已经编好的车组进行调度安排车次）
   ⑤车票出售
   ⑥车票退票
   ⑦根据车次或者乘车人证件号查询车票
   ⑧在主界面进行各种查询
